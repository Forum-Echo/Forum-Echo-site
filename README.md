# [ForumEcho](https://forumecho.eu) - Site

This project is the front-end of the ForumEcho platform.

## Technology

We are using the Angular Framework by Google

## Features

- Homepage
    - all posts
    - Sorter
- Login
- Register
- New Post
- Edit Post
- AboutPage
- Forgot-Password
- User-Settings
